# Order Management API

Este é um projeto de API de gestão de pedidos para um e-commerce. A API foi desenvolvida em C# com .NET 8.0 e utiliza autenticação JWT.

## Funcionalidades

- Cadastro de usuários
- Registro de produtos
- Criação de pedidos
- Autenticação JWT

## Tecnologias

- C#
- .NET 8.0
- PostgreSQL
- Docker
- Swagger
- JWT Authentication
